API REST Spring Boot (Java 17) avec Jakarta (jakarta.*), Spring MVC, Spring Data JPA et Lombok.

## Stack
- Spring Boot Web, Data JPA, Validation, Security, Thymeleaf
- Flyway (migrations)
- MapStruct (mappers), Lombok
- DB:  MySQL
- Build: Maven

## Variables d’environnement utiles
- SPRING_PROFILES_ACTIVE
- DB_USERNAME / DB_PASSWORD


## Comptes par défaut
- Rôle ADMIN
    - username: admin
    - password: demo1234

Utilisez ces identifiants pour vous connecter avec les privilèges administrateur sur USER.
